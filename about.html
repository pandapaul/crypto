Crypto

Crypto creates encrypted messages using randomly generated substitution ciphers. The encryption is simple enough that the messages can be decrypted by an individual person given a single hint. An interesting thing about Crypto, though, is that the messages are live. This means you can send the message URL to a whole bunch of people, who can all compete and get in each other's way... or work together to solve it.

Node.js, Express, and jQuery were used to create Crypto, which is an open source project hosted on GitHub and deployed to Heroku.
